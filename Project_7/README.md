## آزمایش7

## گزارش کار
هدف این آزمایش ساخت یک ماشین حساب ساده است که چهار عمل اصلی ریاضی را انجام دهد. ورودی نمایشگر سریال به صورت عدد، عملگر، عدد خواهد بود. برنامه باید:

1. عملوند اول را بخواند.
2. سپس عملگر را بخواند.
3. نهایتاً عملوند دوم را بخواند.

برنامه سپس عملگر را تفسیر کرده و محاسبات لازم را انجام می‌دهد. برای عملیات تقسیم، مراقب تقسیم بر صفر هستیم و اگر عملوند دوم صفر باشد، محاسبه را متوقف می‌کنیم.

---

### کد برنامه
این کد یک ماشین حساب ساده برای آردوینو است که ورودی‌های عددی و عملگرهای ریاضی را از طریق نمایشگر سریال می‌گیرد و نتیجه را محاسبه و نمایش می‌دهد.

1. چهار متغیر برای ذخیره اعداد، عملگر و نتیجه تعریف شده‌اند.
2. در تابع setup، ارتباط سریال با نرخ ۹۶۰۰ بیت در ثانیه آغاز می‌شود.
3. در تابع loop، اگه ورودی سریال وجود داشته باشه، عدد اول، عملگر و عدد دوم خونده می‌شن.
4. براساس عملگر خوانده شده (+, -, *, /)، عملیات ریاضی مناسب انجام می‌شود.
5. نتیجه محاسبه شده روی نمایشگر سریال چاپ می‌شود. اگر عملگر / باشه و عدد دوم صفر نباشه، تقسیم انجام می‌شه.

به این ترتیب، این برنامه ورودی‌های ریاضی رو پردازش و نتیجه رو نمایش می‌ده.


```cpp
float num1;
float num2;
char operator;
float result;

void setup() {
  Serial.begin(9600);
}

void loop() {
  if (Serial.available() > 0) {
    num1 = Serial.parseFloat(); 
    operator = Serial.read();   
    num2 = Serial.parseFloat(); 

  switch(operator) {
    case '+' : result = num1 + num2;
    break;
    case '-' : result = num1 - num2;
    break;
    case '*' : result = num1 * num2;
    break;
    case '/' :
    if (num2 != 0) {    
     result = num1 / num2;
    }
    break;
  }
  
   Serial.print(result);    
  }
}
```
